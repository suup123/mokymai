      # Ciklo while užduotis nr. 3

      # Užduoties sąlyga

      Išveskite visus skaičius nuo 1 iki 100, kiekvieną skaičių atskiriant kableliu, bei nekeliant skaičių į naujas eilutes.

      # Sprendimas

      Kodas:

      #include <iostream>
using namespace std;

int main()
{
	int skaicius = 1;

	while (skaicius < 100)
	{
		cout << skaicius << ", ";
		skaicius++;
	}

	cout << endl;

	return 0;
}
